# 介绍

这个是 vuepress 的圣诞插件，idea 来源于 https://codepen.io/kevinjannis/pen/Krids (opens new window) 以及 https://codepen.io/xxholly32/details/XWjambb (opens new window)

圣诞节快乐 🎄

# 安装

yarn add -D vuepress-plugin-christmas

还需要配置 sass 相关，安装 sass

yarn add -D sass sass-loader

.vuepress/config.js 配置依赖

module.exports = {
  configureWebpack: {
    module: {
      rules: [
        {
          test: /\.scss$/,
          loader: "sass-loader",
          options: {
            implementation: require("sass"),
          },
        },
      ],
    },
  },
};

# 使用

module.exports = {
  plugins: ["vuepress-plugin-christmas"],
};
